# Ivy

A local podcast and audiobook player for Android with library management, clips with automatic transcription and innovative player UI.

### AI Notice

This project was created with assistance from Claude.


## Development Setup

### Prerequisites

- Node.js
- **Android**: Android Studio with SDK installed, `ANDROID_HOME` environment variable set

### Install Dependencies

```bash
npm install
```

### Run Development Client

```bash
npm start # prebuilds, builds APK and installs via adb
```


## Production Builds

Build the APK with:

```bash
npm run build
```
